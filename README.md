## Usage
Clone it by
```bash
git clone https://github.com/anubhavanonymous/Dark_Eagle
```
Go to the Dark Eagle Directory
```bash
cd Dark_Eagle
```
Install requirements 
```bash
bash setup.sh
```
Run Dark Eagle 
```bash
bash eagle.sh
```
